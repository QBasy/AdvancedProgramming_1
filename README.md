## How to start server:
1. Build: `go build -tags netgo -ldflags '-s -w' -o main`
2. Start: `go run main.go`

## Used Instruments:
- Backend: Go, Gorm, Gomail, Logrus
- Frontend: HTML + CSS + BOOTSTRAP + Vanilla JavaScript
- DataBase: PostgreSQL
